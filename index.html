QS 프로젝트 비교표

현재 프로젝트와 유사 프로젝트 3개의 층별총집계표를 업로드하여 아이템 통일 후 동별 비교표를 생성합니다.

1. 분석표 데이터 업로드

프로젝트별 층별총집계표 EXCEL 파일을 업로드합니다.

현재 프로젝트 여러 파일 가능
A 프로젝트 여러 파일 가능
B 프로젝트 여러 파일 가능
C 프로젝트 여러 파일 가능

2. 아이템 통일 설정

유사한 아이템끼리 자동으로 묶은 뒤, 그룹 단위로 통일 아이템명을 설정합니다.

현재 프로젝트
A 프로젝트
B 프로젝트
C 프로젝트
통일 아이템명

3. 동별 비교표

동을 선택하면 아이템별 카드로 현재 프로젝트, 유사 3개 프로젝트, 평균, 비율을 층별 비교합니다.
