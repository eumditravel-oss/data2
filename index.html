QS 프로젝트 비교표

현재 프로젝트와 유사 프로젝트 3개의 층별총집계표를 업로드하여 동별 비교표를 생성합니다.

1. 분석표 데이터 업로드

프로젝트별 층별총집계표 EXCEL 파일을 업로드합니다.

현재 프로젝트 여러 파일 가능
A 프로젝트 여러 파일 가능
B 프로젝트 여러 파일 가능
C 프로젝트 여러 파일 가능

2. 아이템 통일 설정

프로젝트별 3행/4행 아이템명을 비교한 뒤, 통일할 명칭을 설정합니다.

프로젝트 원본 아이템명 자동 제안명 통일 아이템명
업로드 후 아이템을 추출하면 여기에 표시됩니다.

3. 동별 비교표

동을 선택하면 현재 프로젝트와 유사 3개 프로젝트, 평균, 비율을 층별로 비교합니다.

비교할 데이터를 먼저 준비해 주세요.
