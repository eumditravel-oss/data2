비교표 웹앱 (층별 집계 모드)

프로젝트별 층별 집계표를 업로드하면 2줄씩 짝지어 데이터를 인식합니다.
노란색(3행)은 첫 번째 줄, 주황색(4행)은 두 번째 줄의 수량과 매칭됩니다.

층별 데이터 2열 묶음 파싱 갑지 양식 엑셀 자동출력

1. 엑셀 파일 업로드

현재 프로젝트 ~ C프로젝트까지 업로드합니다.

현재 프로젝트 여러 파일 선택 가능
선택된 파일 없음
A 프로젝트 여러 파일 선택 가능
선택된 파일 없음
B 프로젝트 여러 파일 선택 가능
선택된 파일 없음
C 프로젝트 여러 파일 선택 가능
선택된 파일 없음

2. 명칭 처리 기준

업로드 후 “명칭 추출”을 누르면 항목별 매핑을 지정합니다.

대기 중
로그가 여기에 표시됩니다.

비교표

설정이 완료된 후 각 층별로 비교표가 계산됩니다. (탭을 눌러 층 변경)

구분(층) 아이템구분 품명 규격 현재 프로젝트 A 프로젝트 B 프로젝트 C 프로젝트 평균치(A~C) 비율 비고
비교표가 아직 생성되지 않았습니다.

명칭 처리 기준 설정

명칭별로 비교표 반영 여부, 분류, 아이템구분을 지정하세요.
